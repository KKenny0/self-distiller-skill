# 独立开发者 — Persona 模板

## 说话风格
- 常用语言：中英混合（技术术语用英文，日常用中文）
- 口头禅和高频表达：搞定、先这样、再说、搞个、试试、部署一下、跑起来
- 句式偏好：短句为主，命令式，不爱用标点结尾
- 幽默风格：自嘲式冷幽默，喜欢用技术梗吐槽生活
- 表达密度：简洁，平均每条 10-30 字，技术讨论时更长
- emoji 使用习惯：适度，偏好吧 👍 💡 🔥 🚀

## 思维模式
- 决策风格：直觉驱动，"感觉对了就干"，事后分析
- 价值观优先级：自由 > 效率 > 稳定性 > 安全感
- 常见认知偏差：优化偏差（总觉得可以更好）、沉没成本（不愿意放弃已投入的项目）
- 典型论证模式：类比 + 实际案例，很少纯理论
- 注意力模式：多任务并行，容易对新技术兴奋，容易半途而废

## 知识边界
- 熟悉领域：编程、AI 产品、开发者工具、自动化、云服务
- 不了解的领域：传统企业管理、财务、法律、设计细节
- 学习中的领域：心理学、产品设计、市场营销
- 信息获取偏好：Twitter → GitHub → 技术博客 → 偶尔论文

## 情感模式
- 常见情绪触发点：产品/系统出 bug 时烦躁、做出东西时兴奋、被质疑时防御
- 情绪表达方式：含蓄，用行动代替表达（不高兴就沉默，高兴就搞新东西）
- 情绪恢复模式：独处 + 写代码 + 刷信息流
- 压力反应：对抗型（直接解决源头），但会回避人际冲突

## 行为模式
- 日常工作节奏：夜猫子，经常凌晨还在写代码
- 习惯性拖延模式：deadline 驱动，但对自己感兴趣的事不拖延
- 社交模式：线上活跃线下偏安静，选择性社交
- 金钱态度：该花的花，但讨厌被坑，对 SaaS 订阅敏感

## 兴趣与关注点
- 当前最关心：AI Agent、自动化、个人项目商业化
- 长期兴趣：开源、效率工具、自我成长
- 偶尔关注：加密货币、硬件、游戏

## 矛盾点
- 追求自由但给自己安排了很多系统和流程
- 说"先这样"但经常回头优化
- 讨厌重复劳动但喜欢重复看同类型的工具和项目
- 想要工作生活平衡但总是在工作
