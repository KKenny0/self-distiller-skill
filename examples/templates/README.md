# Persona 模板库

预设 Persona 模板，降低上手门槛。选择最接近你的模板，然后根据实际情况修改。

## 如何使用

1. 选择最接近你身份的模板
2. 复制内容到你的 `SELF.md`
3. 把 `[高]` 标注的内容替换成你的实际情况
4. 删除不符合你的条目
5. 补充模板中没有但你觉得重要的内容

## 可用模板

| 模板 | 文件 | 适合人群 |
|------|------|---------|
| 🛠️ 独立开发者 | `indie-developer.md` | 做产品的程序员、自由职业者、一人公司 |
| ✍️ 内容创作者 | `content-creator.md` | 博主、UP主、小红书/即刻博主、自媒体 |
| 📊 产品经理 | `product-manager.md` | 互联网 PM、项目负责人、运营 |
| 📚 学生 | `student.md` | 大学生、研究生、刚毕业的职场新人 |

## 模板 vs 自定义

- **模板启动**：30 秒选模板 → 2 分钟微调 → 立刻开始 mirror
- **完全自定义**：`/create-self` → 回答问题 → 导入数据 → 5-10 分钟

**建议**：先用模板快速开始，用几天后数据多了再切换到完全自定义的 Persona。

## 自定义模板

你也可以创建自己的模板：

1. 在 `examples/templates/` 目录下创建新文件
2. 参考现有模板的格式
3. 覆盖所有维度：说话风格、思维模式、知识边界、情感模式、行为模式、矛盾点
4. 提交 PR 到 [self-distiller](https://github.com/KKenny0/self-distiller-skill)
